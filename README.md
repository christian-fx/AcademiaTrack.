# AcademiaTrack

**AcademiaTrack** is a web-based CGPA calculator designed to help students track and manage their academic performance efficiently. Users can sign up using email or Google, calculate their semester or cumulative GPA, and securely store their results online using Firebase.

## Features

- **Signup/Login:** Email/password and Google authentication  
- **Secure Storage:** CGPA data stored in Firebase Firestore  
- **Cross-Device Access:** Works on mobile and desktop  
- **Prevent Duplicate Accounts:** Ensures unique emails  
- **User-Friendly Interface:** Simple, intuitive navigation  

## Technologies Used

- HTML, CSS, JavaScript  
- Firebase Authentication & Firestore  
- Hosted on vercel

## How to Use

1. Open the landing page (`index.html`)  
2. Sign up or log in  
3. Calculate your CGPA  
4. Save and view your results securely
